# Xcode Themes

A collection of custom Xcode color themes.

## Themes

- **Catppuccin** — Frappé, Latte, Macchiato, Mocha
- **Dracula** — Original and customized variants
- **One Dark**
- **Space Grey** — Multiple variants
- **eppz!** — Original and customized

## Install

Copy the theme files to your Xcode themes directory:

```bash
cp *.xccolortheme *.dvtcolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
```

Restart Xcode, then select a theme under **Xcode → Settings → Themes**.
